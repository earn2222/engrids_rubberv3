# คู่มือการใช้งาน ระบบจัดการแปลงยางพารา
**URL:** http://localhost:3499/rub/

---

## ขั้นตอนเริ่มต้น — เข้าสู่ระบบ

1. เปิดเบราว์เซอร์ไปที่ `http://localhost:3499/rub/`
2. กดปุ่ม **"Sign in with Google"** มุมบนขวา
3. เลือกบัญชี Google ของคุณ
4. เมื่อเข้าสู่ระบบสำเร็จ — รูปโปรไฟล์และชื่อจะแสดงแทนปุ่ม Sign in
5. หน้าจะโหลดรายการโครงการและทีมงานโดยอัตโนมัติ

> **สำคัญ:** ต้อง Login ก่อนทุกครั้ง มิเช่นนั้นจะเข้าหน้า Reshape และ Dashboard ไม่ได้

## โมดูล 1 — Reshape (ปรับรูปแปลงโฉนด)

**วัตถุประสงค์:** ปรับขอบเขตรูปแปลงโฉนดให้ตรงกับพื้นที่จริง เนื้อที่ขณะนี้ต้องใกล้เคียงเนื้อที่เป้าหมาย (ต่างกันไม่เกิน 100 m²)

### วิธีเข้าหน้า Reshape
1. หน้าหลัก → กดปุ่ม **"ปรับรูปแปลง"** ในโครงการที่ต้องการ
2. เลือกชื่อของคุณในกล่อง Modal ที่เด้งขึ้นมา
   - ชื่อที่ตรงกับ Login จะถูก highlight สีเขียว
   - กด **"ดูทั้งหมด"** ถ้าต้องการดูแปลงทั้งโครงการ
3. ระบบจะนำไปยังหน้า Reshape เฉพาะช่วง ID ที่คุณรับผิดชอบ

### ทำความเข้าใจหน้า Reshape

**แผนที่ (ซ้าย)**
- แปลงสีเขียว = เนื้อที่ถูกต้องแล้ว (ต่างจากเป้าหมาย ≤ 100 m²)
- แปลงสีส้ม = เนื้อที่ยังไม่ถูกต้อง ต้องปรับ
- แปลงที่เลือก = เส้นขอบเปลี่ยนเป็นสีฟ้าสว่าง

**แผงข้อมูล (ขวา)**
- **ID** — รหัสแปลง
- **เลขทะเบียนเกษตรกร** — รหัสเกษตรกรเจ้าของแปลง
- **เนื้อที่เป้าหมายโฉนด (m²)** — พื้นที่ที่ต้องการให้ได้
- **เนื้อที่ขณะนี้ (m²)** — พื้นที่จริงที่วาดอยู่
- Badge **"เนื้อที่ใกล้เคียงกัน"** (เขียว) หรือ **"เนื้อที่ยังไม่เท่ากัน"** (แดง)

**เครื่องมือวาดบนแผนที่ (แถบด้านขวาของแผนที่)**
- ไอคอนรูปหลายเหลี่ยม — วาด Polygon ใหม่
- ไอคอนดินสอ — แก้ไขรูปทรงที่มีอยู่
- คลิกขวาที่จุดบนแผนที่ → ลบจุดนั้น

---

### ขั้นตอนการปรับรูปแปลง

#### กรณี A — แปลงมี Polygon อยู่แล้ว (ต้องการปรับรูปทรง)

**ขั้นตอนที่ 1:** เลือกแปลงที่ต้องการปรับ
- คลิกที่แปลงบนแผนที่ → แปลงเปลี่ยนเป็นสีฟ้า ข้อมูลโหลดในแผงขวา
- หรือในตาราง (ด้านล่างสุด) กดปุ่ม **"ซูม"** เพื่อ Zoom ไปยังแปลงนั้น

**ขั้นตอนที่ 2:** ตรวจสอบข้อมูล
- ดูค่า **เนื้อที่เป้าหมาย** กับ **เนื้อที่ขณะนี้**
- ถ้า badge แสดง **"เนื้อที่ยังไม่เท่ากัน"** → ต้องปรับรูปแปลง

**ขั้นตอนที่ 3:** เปิดโหมดแก้ไข
- กดไอคอน **ดินสอ (Edit)** บนแถบเครื่องมือขวาของแผนที่
- หรือคลิกที่แปลงที่เลือกแล้ว — จุด Vertex จะแสดงขึ้นมาให้ลาก

**ขั้นตอนที่ 4:** ปรับรูปแปลง
- **ลากจุด Vertex** (จุดกลม) เพื่อเปลี่ยนมุมแปลง
- **ลากกึ่งกลางขอบ** (จุดเล็ก) เพื่อเพิ่มจุดใหม่
- **คลิกขวาที่จุด** เพื่อลบจุดนั้นออก
- ค่า **เนื้อที่ขณะนี้** จะอัปเดตทันทีขณะลาก
- ปรับจนกว่า badge จะเปลี่ยนเป็น **"เนื้อที่ใกล้เคียงกัน"** (เขียว)

**ขั้นตอนที่ 5:** บันทึก
- กดปุ่ม **"Save"** ในแผงขวา
- รอจนกว่าจะแจ้ง "อัพเดท features เรียบร้อย"
- แผนที่จะโหลดใหม่และแสดงแปลงที่ปรับแล้ว

---

#### กรณี B — แปลงเป็น Point (ยังไม่มี Polygon)

แปลงที่เป็น Point จะแสดงเป็นไอคอนต้นยาง ต้องวาด Polygon ทดแทน

**ขั้นตอนที่ 1:** คลิกที่ไอคอนต้นยางบนแผนที่
- ไอคอนจะเปลี่ยนเป็นสีฟ้า ข้อมูลโหลดในแผงขวา

**ขั้นตอนที่ 2:** วาด Polygon ใหม่
- กดไอคอน **วาด Polygon** (รูปหลายเหลี่ยม) บนแถบเครื่องมือขวาของแผนที่
- คลิกบนแผนที่เพื่อวางจุดทีละจุดตามขอบเขตแปลง
- **ดับเบิลคลิก** หรือ **คลิกจุดแรก** เพื่อปิดรูปทรงและจบการวาด
- ระบบจะคำนวณเนื้อที่และแสดงผลใน badge อัตโนมัติ

**ขั้นตอนที่ 3:** ปรับแต่งและบันทึก
- หากต้องการแก้ไขรูปทรง ลากจุด Vertex ได้เลย
- กดปุ่ม **"Save"** เมื่อพอใจกับรูปทรงแล้ว

---

#### กรณี C — Restore แปลงกลับเป็นค่าเดิม

ใช้เมื่อต้องการย้อนกลับแปลงที่เคยแก้ไขไปแล้ว

**ขั้นตอนที่ 1:** กดปุ่ม **"restore"** ในแผงขวา
**ขั้นตอนที่ 2:** ใส่ **ID** ของแปลงที่ต้องการ Restore ในช่อง Input
**ขั้นตอนที่ 3:** กดปุ่ม **"Restore please"**
**ขั้นตอนที่ 4:** รอแจ้งว่า "Restore เรียบร้อย" แปลงจะกลับเป็นรูปทรงเดิม

---

### ไปหน้า Reclass จากหน้า Reshape

เมื่อปรับรูปแปลงเสร็จแล้ว → กดปุ่ม **"Classify แปลงที่นี้"**
- ระบบจะสร้าง Reclass Layer ให้อัตโนมัติแล้วพาไปหน้า Reclass

## โมดูล 2 — Reclass (จำแนกประเภทแปลง)

**วัตถุประสงค์:** จำแนกประเภทพื้นที่ภายในแปลง เช่น ส่วนไหนเป็นยางพารา ส่วนไหนเป็นบ่อน้ำ หรือสิ่งปลูกสร้าง

### วิธีเข้าหน้า Reclass
- จากหน้า Reshape → กด **"Classify แปลงที่นี้"** (แนะนำ)
- จากหน้า Dashboard (Reclassdash) → ตาราง → กดปุ่ม **"แก้ไข"** ในแถว

### สีแทนประเภทบนแผนที่

| สี | รหัส | ความหมาย |
|----|------|---------|
| เขียวเข้ม | rubber | ยางพาราที่ลงทะเบียน |
| ม่วง | not-rubber | ยางพาราที่ไม่ได้ลงทะเบียน |
| แดง | Other | ไม่ใช่ยางพารา |
| เขียวสด | ex_age_rubber | พื้นที่กันออก — ยางต่างอายุ |
| ชมพู | ex_building | พื้นที่กันออก — สิ่งปลูกสร้าง |
| ฟ้าสว่าง | ex_pond | พื้นที่กันออก — บ่อน้ำ |
| เหลือง | ex_cr_area | พื้นที่กันออก — ถนนคอนกรีต |
| น้ำเงินเข้ม | ex_ar_area | พื้นที่กันออก — ถนนลาดยาง |
| ส้ม | ex_other | พื้นที่กันออก — เพิ่มเติม |

### ขั้นตอนการ Reclass

#### กรณี A — เปลี่ยนประเภทแปลง (ไม่ต้องตัดแปลง)

ใช้เมื่อทั้งแปลงมีประเภทเดียว เช่น เป็นยางทั้งหมด

**ขั้นตอนที่ 1:** คลิกที่แปลงบนแผนที่
- แปลงที่เลือกจะเปลี่ยนสี ข้อมูลโหลดในแผงขวา (ID, Sub ID, เลขทะเบียน, เนื้อที่)

**ขั้นตอนที่ 2:** เลือกประเภทจาก Dropdown **"ประเภท"** ในแผงขวา
- เลือกประเภทที่ถูกต้อง เช่น "ยางพาราที่ลงทะเบียน"

**ขั้นตอนที่ 3:** กดปุ่ม **"Save"**
- แปลงจะเปลี่ยนสีบนแผนที่ตามประเภทที่เลือก

---

#### กรณี B — ตัดแปลง (Split) เพื่อแบ่งเป็น 2 ส่วน

ใช้เมื่อในแปลงเดียวมีหลายประเภท เช่น บางส่วนยาง บางส่วนบ่อน้ำ

**ขั้นตอนที่ 1:** คลิกเลือกแปลงที่ต้องการตัด

**ขั้นตอนที่ 2:** กดปุ่ม **"วาดเส้นตัด"** บนแถบเครื่องมือขวาของแผนที่
- Hint Bar สีเทาจะแสดงขึ้นบนแผนที่ว่า "กำลังวาดเส้นตัด"

**ขั้นตอนที่ 3:** วาดเส้นตัดผ่านแปลง
- คลิกบนแผนที่เพื่อวางจุดแรก → ลากเส้นผ่านแปลง → คลิกวางจุดต่อไป
- **ดับเบิลคลิก** เพื่อจบเส้นตัด
- เส้นตัดต้องผ่านขอบแปลงทั้ง 2 ด้าน (เข้าด้านหนึ่ง ออกอีกด้านหนึ่ง)
- สามารถลากจุดบนเส้นตัดเพื่อปรับตำแหน่งได้

**ขั้นตอนที่ 4:** กดปุ่ม **"ยืนยันตัดแปลง"**
- แปลงจะถูกตัดเป็น 2 ชิ้น แต่ละชิ้นจะกลายเป็น Sub แปลงใหม่

**ขั้นตอนที่ 5:** ตั้งประเภทให้แต่ละชิ้น
- คลิกชิ้นแรก → เลือกประเภท → กด Save
- คลิกชิ้นที่สอง → เลือกประเภท → กด Save

> **ยกเลิก:** กดปุ่ม **"ยกเลิก"** บน Hint Bar หรือกดปุ่ม **"Clear"** ในแผงขวาเพื่อล้างเส้นตัดออก

---

#### กรณี C — ตัดแปลงหลายส่วน

วาดเส้นตัดใหม่ซ้ำๆ เพื่อแบ่งเป็น 3, 4 ส่วน โดยทำทีละเส้นตามขั้นตอน B

---

#### กรณี D — คืนแปลงเดิม (Unsplit)

ใช้เมื่อต้องการยกเลิกการตัดและรวม Sub แปลงกลับเป็นแปลงเดิม

**ขั้นตอนที่ 1:** คลิกเลือกแปลงที่ต้องการคืน

**ขั้นตอนที่ 2:** กดปุ่ม **"คืนแปลงเดิม"** ในแผงขวา

**ขั้นตอนที่ 3:** ยืนยัน → ระบบจะรวม Sub แปลงทั้งหมดกลับเป็นแปลงเดิม

---

#### กรณี E — ปรับรูปทรง Polygon (Edit)

ใช้เมื่อต้องการขยับหรือปรับขอบแปลงใน Reclass

**ขั้นตอนที่ 1:** คลิกเลือกแปลงที่ต้องการแก้ไข

**ขั้นตอนที่ 2:** กดไอคอน **แก้ไข (Edit)** บนแถบเครื่องมือขวาของแผนที่
- Edit Hint Bar จะแสดงขึ้น: "โหมดแก้ไข — ลากจุดยอด / ขอบเพื่อเปลี่ยนรูปแปลง"

**ขั้นตอนที่ 3:** ลากจุดหรือขอบแปลงเพื่อปรับรูปทรง

**ขั้นตอนที่ 4:** กด **"ยกเลิก"** บน Hint Bar เพื่อออกจากโหมดแก้ไข (ไม่บันทึก)
หรือกด **"Save"** เพื่อบันทึกรูปทรงใหม่

---

#### กรณี F — Merge Polygon (รวมหลายแปลงเป็นหนึ่ง)

ใช้เมื่อต้องการรวม 2 Sub แปลงขึ้นไปเข้าด้วยกัน

**ขั้นตอนที่ 1:** กดปุ่ม **"เข้าโหมด Merge"** ในแผงขวา
- ส่วน Merge Panel จะแสดงขึ้น

**ขั้นตอนที่ 2:** คลิกที่แปลงบนแผนที่ทีละอัน (เลือกอย่างน้อย 2 อัน)
- รายชื่อแปลงที่เลือกจะปรากฏใน "selectedPolygonsList"

**ขั้นตอนที่ 3:** เมื่อเลือกครบแล้ว กดปุ่ม **"Merge (เลือก 2 ตัวขึ้นไป)"**
- แปลงทั้งหมดจะรวมเป็น Polygon เดียว

**ขั้นตอนที่ 4:** กด **"ออกจากโหมด Merge"** เพื่อกลับโหมดปกติ

## โมดูล 3 — Reclassdash (Dashboard ตรวจสอบ)

**วัตถุประสงค์:** ตรวจสอบผลการ Reclass ของแปลงทั้งหมด บันทึกผล ผ่าน/ไม่ผ่าน และหมายเหตุ

### วิธีเข้าหน้า Reclassdash
1. หน้าหลัก → กดปุ่ม **"Dashboard"** ในโครงการที่ต้องการ
2. เลือกชื่อตัวเองในกล่อง Modal
3. ระบบจะแสดงเฉพาะแปลงในช่วง ID ที่คุณรับผิดชอบ

---

### ส่วนที่ 1 — แผนที่ (ซ้าย)

**เปลี่ยนแผนที่พื้นหลัง:**
- กดไอคอน Layer Control มุมบนขวาของแผนที่
- เลือก: Google Road / **Google Satellite** (ค่าเริ่มต้น) / Google Terrain / Google Hybrid / Stadia Light

**เปิด/ปิด Layer:**
- แปลงยาง (reclass) — แปลงที่จำแนกแล้ว สีตามประเภท
- แปลงยาง (reshape) — แปลงหลังปรับรูป
- แปลงยาง (เดิม) — แปลงต้นฉบับ (โหลดตามพื้นที่ที่มองเห็น)
- Longdo Map — ชั้นแผนที่ไทย

**การโต้ตอบ:**
- คลิกที่แปลงบนแผนที่ → เปิด Popup แสดง ID + โหลดข้อมูลในแผงขวาทันที
- แผนที่จะ Zoom เข้าไปที่แปลงที่เลือกโดยอัตโนมัติ

---

### ส่วนที่ 2 — แผงข้อมูลแปลง (ขวา)

#### 2.1 ค้นหาแปลง

**ขั้นตอนที่ 1:** พิมพ์ ID หรือเลขทะเบียนเกษตรกรในช่องค้นหา
**ขั้นตอนที่ 2:** กด Enter หรือกดปุ่ม **แว่นขยาย**
- แผนที่จะ Zoom ไปที่แปลง + ข้อมูลโหลดในแผง

#### 2.2 เลื่อนดูแปลงทีละอัน

- กดปุ่ม **"<"** — ไปแปลงก่อนหน้า
- กดปุ่ม **">"** — ไปแปลงถัดไป
- ตัวเลข **"x / y"** ตรงกลาง — บอกว่าอยู่แปลงที่เท่าไหร่จากทั้งหมด
- กดที่ตัวเลข — กลับมา Focus ที่แปลงปัจจุบันบนแผนที่

---

### ส่วนที่ 3 — กล่องเจ้าหน้าที่ Checker

กล่องสีเขียว ใช้สำหรับ **เจ้าหน้าที่ผู้ตรวจสอบ** บันทึกผลการตรวจ

#### ขั้นตอนการตรวจสอบแปลง

**ขั้นตอนที่ 1:** คลิกเลือกแปลงที่ต้องการตรวจ (จากแผนที่, ตาราง หรือปุ่ม </>)

**ขั้นตอนที่ 2:** ดูข้อมูลแปลง
- เปรียบเทียบ **เนื้อที่เป้าหมายโฉนด** กับ **เนื้อที่ขณะนี้**
- เปรียบเทียบ **เนื้อที่เป้าหมายยางพารา** กับ **เนื้อที่คลาส**
- ดู **ประเภทปัจจุบัน** (Badge สี) ว่าถูกต้องหรือไม่

**ขั้นตอนที่ 3:** ระบุผลการตรวจ
- **ตรวจสอบโฉนด** → เลือก `✅ ผ่าน` หรือ `❌ ไม่ผ่าน`
- **การจำแนกประเภท** → เลือก `✅ ผ่าน` หรือ `❌ ไม่ผ่าน`

**ขั้นตอนที่ 4:** พิมพ์หมายเหตุ (ถ้ามี)
- ช่อง **"หมายเหตุ / ความเห็น"** — ระบุรายละเอียดปัญหา
- พิมพ์แบบมีลำดับได้ เช่น `1. ขอบแปลงเกิน 2. ประเภทไม่ถูก`

**ขั้นตอนที่ 5:** กดปุ่ม **"บันทึกผลการตรวจ"**
- ปุ่มจะเปลี่ยนเป็น "เรียบร้อย" เมื่อบันทึกสำเร็จ
- แถวในตารางจะเปลี่ยนสี: เขียว = ผ่านทั้งหมด, แดง = มีไม่ผ่าน
- Badge สถานะบนกล่องจะอัปเดตทันที

> **Draft (ร่าง):** เมื่อเปลี่ยนค่าแต่ยังไม่กด "บันทึก" จะมี Badge สีเหลือง **"ร่าง"**
> กดปุ่ม **↩** เพื่อยกเลิกและคืนค่าเดิม

---

### ส่วนที่ 4 — กล่องผู้ใช้งาน (User)

กล่องสีน้ำเงิน ใช้สำหรับ **ผู้ใช้งานทั่วไป** แจ้งสถานะการแก้ไข

**ขั้นตอนที่ 1:** พิมพ์ข้อความในช่อง **"หมายเหตุ / แจ้งการแก้ไข"**
- เช่น "แก้ไขแล้ว" หรือ "ปรับเส้นแปลงแล้วเมื่อ 14/06/2026"

**ขั้นตอนที่ 2:** กดปุ่ม **"บันทึกหมายเหตุ"**

**ขั้นตอนที่ 3:** กดปุ่ม **ถังขยะ** เพื่อลบหมายเหตุ (ต้องยืนยัน)

> **Auto-save:** ระบบจะบันทึกหมายเหตุผู้ใช้อัตโนมัติเมื่อคุณเปลี่ยนไปดูแปลงอื่น

---

### ส่วนที่ 5 — ตารางข้อมูลแปลงทั้งหมด (ด้านล่างสุด)

#### การกรองแปลง

กดปุ่มกรองที่ด้านบนตาราง:

| ปุ่ม | แสดง |
|-----|------|
| ทั้งหมด | แปลงทุกอัน |
| ⏳ ยังไม่ตรวจ | แปลงที่ยังไม่มีการบันทึกผล |
| ✅ ผ่านทั้งหมด | แปลงที่ผ่านทั้งโฉนดและประเภท |
| ❌ มีไม่ผ่าน | แปลงที่มีอย่างน้อยหนึ่งรายการไม่ผ่าน |
| 💬 มีหมายเหตุ | แปลงที่มีหมายเหตุจากผู้ตรวจหรือผู้ใช้ |

#### การบันทึกจากตาราง (แบบเร็ว)

**ขั้นตอนที่ 1:** หาแปลงในตาราง (ค้นหาหรือกรอง)
**ขั้นตอนที่ 2:** เปลี่ยน Dropdown "ตรวจสอบโฉนด" และ "ตรวจสอบการจำแนก" ในแถวนั้น
**ขั้นตอนที่ 3:** พิมพ์หมายเหตุในช่องหมายเหตุผู้เช็ค (ถ้ามี)
**ขั้นตอนที่ 4:** กดปุ่ม **"บันทึก"** ในแถวนั้น

#### ดูหมายเหตุเต็ม

- กดปุ่ม **👁 (ตาเล็ก)** ข้างช่องหมายเหตุ → Popup แสดงข้อความเต็ม
- ข้อความที่ขึ้นต้นด้วยตัวเลข (1. 2. 3.) จะแสดงเป็น Ordered List อัตโนมัติ

#### ดูประวัติการตรวจสอบ

**ขั้นตอนที่ 1:** กดปุ่ม **🕐 (นาฬิกา)** คอลัมน์ประวัติในแถวที่ต้องการ
**ขั้นตอนที่ 2:** Modal ประวัติจะแสดงรายการทุกครั้งที่มีการเปลี่ยนแปลง

ประวัติแสดงสาเหตุ:
- 🔄 คืนค่าแปลง
- ✏️ ปรับรูปแปลง
- 🗑️ ล้างข้อมูล
- 🏷️ อัปเดตประเภท
- 📐 อัปเดตรูปทรง
- ✂️ ตัดแบ่งแปลง
- 🔗 ยกเลิกการตัด — คืนเป็นแปลงเดิม

**ขั้นตอนที่ 3:** คลิกที่ข้อความหมายเหตุในประวัติ → Popup แสดงข้อความเต็ม

### ส่วนที่ 6 — รายงานสถิติ

**ขั้นตอนที่ 1:** คลิกที่หัวข้อ **"รายงานสถิติ"** เพื่อขยายส่วนนี้
**ขั้นตอนที่ 2:** ระบบจะแสดงกราฟสรุปจำนวนแปลงและพื้นที่ตามประเภท

## สรุปเวิร์กโฟลว์การทำงาน

```
1. เข้าเว็บ → Login Google
        ↓
2. หน้าหลัก → เลือกโครงการ → กด "ปรับรูปแปลง"
        ↓
3. เลือกชื่อตัวเอง → เข้าหน้า Reshape (เฉพาะ ID ของตน)
        ↓
4. [Reshape] เลือกแปลงสีส้ม → ปรับรูปแปลงให้เนื้อที่ถูกต้อง
        ↓
5. กด "Classify แปลงที่นี้" → เข้าหน้า Reclass
        ↓
6. [Reclass] คลิกเลือกแปลง → เลือกประเภท
   (ถ้าต้องตัดแปลง → วาดเส้นตัด → ยืนยันตัด → ตั้งประเภทแต่ละส่วน)
        ↓
7. กด Save → กด Dashboard → เข้าหน้า Reclassdash
        ↓
8. [Reclassdash] ตรวจสอบแปลงทีละอัน (ใช้ปุ่ม < >)
   → ระบุ ผ่าน/ไม่ผ่าน → กด "บันทึกผลการตรวจ"
        ↓
9. เมื่อตรวจครบ → หน้าหลัก → Download ข้อมูล
```

## Download ข้อมูล (หน้าหลัก)

กดปุ่ม **"Download ข้อมูล"** → เลือกประเภทที่ต้องการ:

| ตัวเลือก | ไฟล์ที่ได้ | รายละเอียด |
|---------|-----------|-----------|
| Download ทั้งหมด | หลายไฟล์พร้อมกัน | ดาวน์โหลดครบทุกประเภทในครั้งเดียว |
| Download แปลงโฉนดของยางพารา | `pacel_yang_<ชื่อโครงการ>.geojson` | ขอบเขตแปลงโฉนด |
| Download reclassify (LU) | `v_reclass_LU_<ชื่อโครงการ>.geojson` | ทุกประเภทการจำแนก |
| Download Reclassify (ยางลงทะเบียน) | `v_reclass_rubber_<ชื่อโครงการ>.geojson` | เฉพาะแปลง rubber |
| Download Reclassify (ยางลงทะเบียน+พื้นที่กันออก) | `v_reclass_rubber_ex_<ชื่อโครงการ>.geojson` | rubber + ex_* ทุกประเภท |
